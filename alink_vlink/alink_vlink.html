What is the "alink" in HTML.

"alink" is the Action Link. If we use "alink" attribute, we can change the color of text(link) when we press on it with cursor. The default color of "alink" is "red".

Code Sample

<body alink="green" vlink="black" link="blue"></body></p>

This is the Action link.

What is the "vlink" in HTML.

"vlink" is the Visited link.
